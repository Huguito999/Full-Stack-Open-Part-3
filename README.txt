The application is deployed at the following link: [Phonebook Backend](https://full-stack-open-part-3-tkrk.onrender.com)

## Endpoints

- `GET /api/persons` - Get all persons.
- `GET /api/persons/:id` - Get a person by ID.
- `POST /api/persons` - Create a new person.
- `DELETE /api/persons/:id` - Delete a person by ID.
- `GET /info` - Get information about the number of people in the phonebook.
